# Tudor Saloon - Premium Barbershop Website

A modern, responsive website for Tudor Saloon barbershop in Iași, Romania.

## Design

- **Colors**: #F29022 Orange, #FFFFFF White, #1A1A1A Dark
- **Fonts**: Anton (Display), Inter (Body)
- **Style**: Magazine-cover aesthetic with bold typography

## Quick Start

```bash
# Open in browser (Python)
python -m http.server 8000

# Or with Node.js
npx http-server

# Visit http://localhost:8000
```
